“第七届百度奖学金博士生专场”参会总结

发表于 2020-01-09 更新于 2020-02-05 分类于 总结 阅读次数：

前言

时光飞逝，转眼间已经到了经Baidu与中国科学院大学联合举办，由读芯术宣传的“AI未来说青年学术论坛”系列讲座的年终特别场，“第七届百度奖学金颁奖典礼暨AI未来说博士生专场”。此次会议于2020年1月5日在百度科技园举办，地点与以往在中关村的中国科学院大学不同。作为连续12场参加了2019年每月举办一次的AI未来说忠实粉丝，借此时机对此系列讲座进行回顾总结。

系列讲座总结

每期分享的专家与主题来自读芯术的整理。

第一期 数据挖掘专场

  1. 李国杰院士：理性认识人工智能的“头雁”作用
  2. 百度熊辉教授：大数据智能化人才管理
  3. 清华唐杰教授：网络表示学习理论及应用
  4. 瑞莱智慧刘强博士：深度学习时代的个性化推荐
  5. 清华柴成亮博士：基于人机协作的数据管理

第二期 自然语言处理专场

  1. 中科院张家俊：面向自然语言生成的同步双向推断模型
  2. 北邮李蕾：关于自动文本摘要的分析与讨论
  3. 百度孙珂：对话技术的产业化应用与问题探讨
  4. 阿里谭继伟：基于序列到序列模型的文本摘要及淘宝的实践
  5. 哈工大刘一佳：通过句法分析看上下文相关词向量

第三期 计算机视觉专场

  1. 北大彭宇新：跨媒体智能分析与应用

  4. 中科院张士峰：基于深度学习的通用物体检测算法对比探索
  5. 港中文李弘扬 ：物体检测最新进展

第四期 语音技术专场

  4. 百度康永国：AI 时代的百度语音技术
  5. 中科院刘斌：基于联合对抗增强训练的鲁棒性端到端语音识别

第五期 量子计算专场

  1. 清华大学翟荟：Discovering Quantum Mechanics with Machine Learning
  2. 南方科技大学鲁大为：量子计算与人工智能的碰撞
  3. 荷兰国家数学和计算机科学中心（CWI）李绎楠：大数据时代下的量子计算
  4. 苏黎世联邦理工学院（ETH）杨宇翔：量子精密测量

第六期 机器学习专场

  1. 中科院张文生：健康医疗大数据时代的认知计算
  2. 中科院庄福振：基于知识共享的机器学习算法研究及应用
  3. 百度胡晓光：飞桨（PaddlePaddle）核心技术与应用实践
  4. 清华大学王奕森：Adversarial Machine Learning: Attack and Defence
  5. 南京大学赵申宜：SCOPE - Scalable Composite Optimization for Learning

  3. 百度朱帆：开放时代的自动驾驶 - 百度Apollo计划
  4. 北理宋文杰：时空域下智能车辆未知区域自主导航技术

第十二期 第七届百度奖学金颁奖典礼暨AI未来说博士生专场

秋招去过百度面试，但是是在上地十街附近，虽然距离中关村软件园ZPARK的百度科技园也不远，但是此次确实是第一次来到传说中的后厂村。周围遍布各大互联网公司，新浪、网易、Neusoft等等，刚好过完元旦，处处布满年味的气息。

使用一句话对整此会议的总结就是：的确很特别，也充满惊喜。

百度奖学金颁奖典礼

百度奖学金在7年内为66人提供1300万，用于资金支持和技术指导。本次为第七届，共有10位海内外博士获奖，每人20万RMB，这对人才培养和科技创新具有重要意义。

茶歇

在短暂的茶歇期间，听取了来自中科院的常建龙博士分享的部分研究内容。首次接触到基于关系的深度学习这一领域，开拓了眼界。个人对于常博士使用小样本分类问题来解决聚类问题的思想十分感兴趣。另外讲真的，百度的蛋糕很好吃。

博士讲座

  • 罗宇男博士：使用AI辅助科学发现，如医药(老药新用)、生物科技以及智慧农业等领域。
  • 董胤蓬博士：对抗样本鲁棒性，主要研究基于黑盒和白盒的攻击方法。
  • 应智韬博士：图神经网络在推荐系统领域的应用，如GraphSAGE算法。
  • 桂韬博士：Social Media领域的自然语言处理技术，对网络语言价值总结为两条：多模态网络语言能够对早期抑郁症发现提供帮助，网络语言交互可以对用户行为进行预测。桂韬博士的GitHub

圆桌讨论

这是我最感兴趣的一part，最大的惊喜是知道了我在微博上关注良久的爱可可-爱生活老师就是来自北邮的陈光老师。虽然我(totally a loser)在学术领域的成就远不及获得百度奖学金获得者，但在陈光老师采访他们的交流过程中，很多处都产生了强烈的共鸣。

  1. 信心、努力、坚持是克服论文0-1质变的三大法宝。
  2. 工作不变，心态改变。(对他人意见的接受力、对自己的看法。)
  3. SOTA并非最终目标，应该端正研究方向。看完论文的“哦，我知道了”和“诶，有点意思”两种感受代表两类论文。
  4. 论文写作的目的是有话要说，而非不得不说。

总结

最大的收获是一个金句：优秀没有统一的标准，却有令人仰望的高度。
